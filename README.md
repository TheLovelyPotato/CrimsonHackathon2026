### Super Ducky Run
CrimsonHackathon2026<br>
The Rubber duckies <br>
A fun Python-based game featuring duckies! Dodge obstacles in an interactive environment using your body movements.<br>

## Installation
1. Clone the repository:
   -git clone https://github.com/TheLovelyPotato/CrimsonHackathon2026.git
2. Navigate to the project folder:
3. Install dependencies:
   - run vitualenv_setup.bat
4. change line 12 in Game.py
   - capture = cv2.VideoCapture(0)
   - 0 for primary camera
   - 1 for secondary camera

## How to Play
* Run the game:
   - python Game.py
* Stand back
* Wait for the camera to initilize 
* Jump or duck to avoid the obsticles

## Features
Real-time camera interaction (jump/duck detection)<br>
Fun, interactive duck-themed gameplay<br>
Obstacle avoidance challenge<br>
Works with both webcam and USB camera<br>
